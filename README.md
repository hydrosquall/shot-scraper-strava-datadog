# shot-scraper-strava-datadog

An archive of running statistics for athletes at Datadog, powered by Github Actions. 

Data is updated weekly in the [datasets](https://github.com/hydrosquall/shot-scraper-strava-datadog/tree/main/datasets) folder.

## Attribution

- This repository uses Simon Willison's [shot-scraper](https://github.com/simonw/shot-scraper) to save an archive of publicly accessible data from Strava.
- Read more about how this works in [Instantly create a GitHub repository to take screenshots of a web page](https://simonwillison.net/2022/Mar/14/shot-scraper-template/).
